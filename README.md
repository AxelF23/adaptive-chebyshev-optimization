# adaptive-chebyshev-optimization
Implementation of Chebyshev acceleration for gradient descent with adaptive step-size.
# Исследование адаптивного выбора шага в градиентных алгоритмах

Этот репозиторий содержит материалы моей курсовой работы на математико-механическом факультете СПбГУ.

Основная цель работы — адаптация Чебышёвского ускорения для оптимизации неквадратичных функций. В проекте исследуется, как реализовать градиентный спуск без гиперпараметров, адаптируя классический метод Чебышёвского ускорения и теорему Янга для неквадратичных функций.

## Эксперименты
### Нелинейный МНК

В файле `Non_linear_Least_Squares_test.ipynb` реализовано применение адаптивного метода к задаче нелинейного метода наименьших квадратов (Non-linear Least Squares). 

#### Описание задачи
Целью является восстановление параметров вектора $x = [a, b, c]$ для модели:
$$y = a \cdot \sin(b \cdot t) + c$$
на основе зашумленных данных.

####  Особенности реализации:
*   **Функция остатков (Residuals):** Алгоритм минимизирует векторную разность между предсказанием модели и данными: $r(x) = f(x, t) - y_{data}$.
*   **Hessian-Free подход:** Оценка кривизны ландшафта (спектра Гессиана) выполняется через **Hessian-Vector Product (HVP)** с использованием конечных разностей. Это позволяет избежать прямого вычисления матрицы вторых производных.
*   **Адаптивность:** Чебышёвский шаг первого порядка $\eta = \frac{2}{\lambda_{max} + \lambda_{min}}$ пересчитывается динамически, что обеспечивает быструю сходимость даже при плохом начальном приближении.

в файле 'Logistic_regression_test' реализовано применение алгоритма к минимизации функции потерь логрегресси с Ridge - регуляризацией и сравнение предложенного алгоритма с классическим градиентным спуском с оптимальным постоянным шагом.
График демонстрирует явное превосходство предлагаемого метода.
