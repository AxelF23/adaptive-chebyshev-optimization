# adaptive-chebyshev-optimization
Implementation of Chebyshev acceleration for gradient descent with adaptive step-size.
# Исследование адаптивного выбора шага в градиентных алгоритмах

Этот репозиторий содержит материалы моей курсовой работы на математико-механическом факультете СПбГУ.

## О проекте
Основная цель работы — адаптация Чебышёвского ускорения для оптимизации неквадратичных функций. В проекте исследуется, как реализовать градиентный спуск без гиперпараметров, адаптируя классический метод Чебышёвского ускорения и теорему Янга для неквадратичных функций.

## Содержание:
* `code/` — реализации алгоритмов на Python.
* `presentation.pdf` — краткий обзор теории и графиков сходимости.
* `paper.pdf` — полный текст работы с доказательствами.

## Эксперименты
В файле `rosenbrock_experiment.ipynb` представлена реализация алгоритма для функции Розенброка высокой размерности ($d=1000$)



## Результаты
Реализованный алгоритм показывает ускоренную сходимость на ряде тестовых функций по сравнению со стандартным градиентным спуском и методом импульса (Momentum).
