# adaptive-chebyshev-optimization
Implementation of Chebyshev acceleration for gradient descent with adaptive step-size.
# Исследование адаптивного выбора шага в градиентных алгоритмах

Этот репозиторий содержит материалы моей курсовой работы на математико-механическом факультете СПбГУ.

## О проекте
Основная цель работы — адаптация Чебышёвского ускорения для оптимизации неквадратичных функций. В проекте исследуется, как реализовать градиентный спуск без гиперпараметров, адаптируя классический метод Чебышёвского ускорения и теорему Янга для неквадратичных функций.

## Эксперименты
В файле `rosenbrock_experiment.ipynb` представлена реализация алгоритма для функции Розенброка высокой размерности ($d=1000$)

В файле `Non_linear_Least_Squares_test.ipynb` реализовано применение адаптивного метода к задаче нелинейного метода наименьших квадратов (Non-linear Least Squares)[cite: 5]. 

#### Описание задачи
Целью является восстановление параметров вектора $x = [a, b, c]$ для модели:
$$y = a \cdot \sin(b \cdot t) + c$$
на основе зашумленных данных[cite: 3, 5].

#### Ключевые особенности реализации:
*   **Функция остатков (Residuals):** Алгоритм минимизирует векторную разность между предсказанием модели и данными: $r(x) = f(x, t) - y_{data}$[cite: 3, 5].
*   **Hessian-Free подход:** Оценка кривизны ландшафта (спектра Гессиана) выполняется через **Hessian-Vector Product (HVP)** с использованием конечных разностей[cite: 3, 4]. Это позволяет избежать прямого вычисления матрицы вторых производных.
*   **Адаптивность:** Чебышёвский шаг первого порядка $\eta = \frac{2}{\lambda_{max} + \lambda_{min}}$ пересчитывается динамически, что обеспечивает быструю сходимость даже при плохом начальном приближении[cite: 3, 4].

#### Результат
Алгоритм успешно сходится к истинным значениям параметров, демонстрируя устойчивость к нелинейности целевой функции и шуму в данных[cite: 5].


## Результаты
Реализованный алгоритм показывает ускоренную сходимость на ряде тестовых функций по сравнению со стандартным градиентным спуском и методом импульса (Momentum).
